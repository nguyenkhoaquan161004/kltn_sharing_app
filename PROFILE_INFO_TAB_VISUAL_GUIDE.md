## ğŸ§® Grid Layout Calculation

### 2-Column Grid
```
Container Width: ~100% (with padding)
Total Width: width - 24px (left) - 24px (right)
Available: width - 48px

Grid Column Width = (Available - Spacing) / 2
Grid Column Width = (Available - 12px) / 2

Example: 360px width
Available: 360 - 48 = 312px
Column: (312 - 12) / 2 = 150px per column
```

- Image Height: 100px (fixed)
